## 1. 产品定位

投研助手是一款面向证券分析师和投研人员的智能研报分析工具，通过AI技术实现研报自动解析、智能问答、多研报对比和股票数据联动，帮助用户将研报阅读时间从2-3小时缩短至分钟级。

**核心价值主张**：让分析师专注于投资决策，而非信息整理。

---

## 2. 目标用户

| 用户角色 | 描述 | 核心诉求 |
|---------|------|---------|
| 高级分析师 | 5年以上经验，覆盖多个行业 | 快速获取关键信息，支持深度研究 |
| 初级分析师 | 1-3年经验，需大量学习 | 降低研报理解门槛，快速上手 |
| 基金经理 | 关注投资组合相关研报 | 多研报对比，发现观点差异 |

---

## 3. 功能需求

### 3.1 研报管理

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:| 
| 研报上传 | 支持PDF/HTML格式上传，单文件≤50MB，支持批量上传 | P0 |
| 研报列表 | 列表展示已上传研报，支持按公司、券商、日期筛选排序 | P0 |
| 研报搜索 | 支持按公司名称、股票代码、券商名称搜索 | P0 |
| 研报删除 | 支持单篇/批量删除研报，删除后关联数据同步清理 | P1 |
| 研报收藏 | 支持收藏研报，快速访问重要文档 | P1 |

### 3.2 研报解析

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:|
| 自动解析 | 上传后自动触发解析，提取结构化信息 | P0 |
| 信息提取 | 提取：标题、分析师、券商、评级、目标价、当前价、核心观点、财务预测 | P0 |
| 解析状态 | 显示解析进度：待解析/解析中/已完成/失败 | P0 |
| 原文预览 | 支持PDF/HTML原文在线预览，保留原始排版 | P0 |
| 手动重试 | 解析失败时支持手动触发重解析 | P1 |

### 3.3 智能问答

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:|
| 单篇问答 | 针对单篇研报进行提问，AI基于内容回答 | P0 |
| 多篇问答 | 支持选择多篇研报，进行跨文档问答 | P0 |
| 原文引用 | 答案中标注引用来源，支持点击跳转到原文位置 | P0 |
| 会话管理 | 保存历史问答记录，支持查看和继续对话 | P0 |
| 推荐问题 | 根据研报内容推荐常见问题，降低使用门槛 | P1 |

### 3.4 多研报对比

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:|
| 对比选择 | 支持选择2-5篇同一公司的研报进行对比 | P0 |
| 自动对齐 | 自动对齐关键字段：评级、目标价、核心观点、财务预测 | P0 |
| 对比视图 | 表格形式展示对比结果，差异项高亮显示 | P0 |
| 导出功能 | 支持导出对比结果为Excel/图片 | P1 |
| 观点总结 | AI自动生成各券商观点差异总结 | P1 |

### 3.5 股票数据

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:|
| 独立查询 | 顶部搜索栏支持股票代码/名称搜索，进入股票详情页 | P0 |
| 研报联动 | 阅读研报时右侧自动展示关联股票数据 | P0 |
| 行情展示 | 展示：最新价、涨跌幅、成交量、市值等基础行情 | P0 |
| 财务指标 | 展示：PE、PB、ROE等关键财务指标 | P0 |
| 相关研报 | 展示该股票的所有已上传研报列表 | P0 |
| 数据刷新 | 行情数据支持手动刷新，自动缓存30秒 | P1 |

### 3.6 标注与收藏

| 功能 | 需求描述 | 优先级 |
|------|---------|:---:|
| 文本标注 | 支持在研报原文中高亮标注，添加笔记 | P1 |
| 标注列表 | 汇总展示所有标注，支持按研报筛选 | P1 |
| 收藏管理 | 管理收藏的研报，支持分类标签 | P1 |

## 4. 非功能需求摘要

| 类别 | 需求 | 目标值 |
|------|------|--------|
| 性能 | 研报解析耗时 | ≤ 30秒（30页PDF） |
| 性能 | 问答响应时间 | ≤ 5秒 |
| 性能 | 页面加载时间 | ≤ 2秒 |
| 可用性 | 系统可用性 | ≥ 99%（工作时间） |
| 安全 | 数据传输 | HTTPS加密 |
| 安全 | 研报存储 | 本地文件系统，不上云 |
